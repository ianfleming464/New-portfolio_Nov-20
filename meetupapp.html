Meetup App (Meetup API)

React, AWS Lambda, CSS, ReCharts, Jest, Puppeteer

My aim was to build a serverless, progressive web application (PWA) which takes info from the Meetup API (and OAuth 2), and displays events in a city of your choice. Created during the Meetup Pro API 30-day trial period, so currently functioning in offline mode.

Building this proved fun and a little challenging. A nice overall introduction to serverless applications, and very satisfying to build a PWA which functions offline, to an extent. Bam. I also enjoyed designing the whole concrete aesthetic.
